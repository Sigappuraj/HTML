HTML Tags Name and Description

Tag Names Description
html This is root element and these tags tells web page is written in HTML5.
title This is title tag of the page and is useful to both search engines and users.
body This tag is used to place all the contents of users.
base Signifies the base URL and specifies the all the relative links in the document.
meta Includes the additional information about the page.
link Creates association with external pages.
style Includes document style information.
script Links to external scripts.
strong Displays text in bold on browsers.
em Displays text in italics on browsers.
cite Highlights the reference information.
del Labels a previously deleted portion of text.
br Creates a line break.
hr Creates horizontal rule and sectional break.
sub Text placed a half-line lower than previous text.
sup Text placed a half-line higher than previous text.
h1 - h6 Six levels of headings on a page.
div Used to Labels a page section.
span Inline lables used to grouping style elements.
p Used to organize paragraphs of text.
img This tag is used to display image files.
map Informs users to interactive image is present with clickable areas.
ol Numbered lists in sequential order.
ul Bullet point list.
li Specify each items of the list.
dl List item definitions.
dt Defines a single term.
dd Description for defined term.
a href="" Creates a hyperlink with defined anchor text.
form This tag is used to create a form.
label What to enter in each field.
fieldset Identifies the group of all fields on form.
input Input attribute shows what type of field information to receive from users.
legend Caption for fieldset element.
select Drop-down box for users to select from options.
optgroup Grouping of entire available options.
option Defines available options in Drop-down list.
button Shows the clickable button for users to submit options.
table Defines all the contents related to table.
caption Description of tables purpose and contain information.
thead The headers that lable the content in each column.
tbody Body of the table data.
tfoot Describes all footer content.
tr content within single row.
th Data in single header item.
td Content within single table cell.
colgroup columns for formatting purpose.
iframe Inline frame that allows external information to be embedded into a document.
param Adds extra parameters help to customize the iframe content.
embed Container for external application or plug-in.
New HTML5 TAGS
header Header block for a document.
main Main content of a document.
aside Content contained in a sidebar.
article Identifies an article within a document.
section Section block in a document.
dialog Box or window for dialog
figure Content block featuring diagrams, photos.
figcaption Figure element caption that describes the figure.
mark Displays highlighted text within the content.
menuitem Menu item user can raise from a popup menu.
meter scalar measurement within a known array.
progress Progress of task, usually used for progress bar.
time Identifies time and date.
